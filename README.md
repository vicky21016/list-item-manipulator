# List Item Manipulator (列表項目操作器)

List Item Manipulator 是一個簡單的網頁應用，允許用戶選擇兩個列表項目並根據選擇的操作方式來調整它們的位置。用戶可以選擇將一個項目放到指定的前面或後面，或者交換兩個選中的項目。這個應用可以幫助演示列表項目的動態操作。

## 功能
- 顯示 1 到 10 的號碼列表，並允許用戶選擇其中兩個項目進行操作。
- 用戶可以選擇不同的操作：
  - **放到指定的後面**：將選中的第二個項目放到第一個項目的後面。
  - **放到指定的前面**：將選中的第二個項目放到第一個項目的前面。
  - **交換**：交換兩個選中項目的位置。
